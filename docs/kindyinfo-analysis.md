# KindyInfo (幼園通) 網站分析

## 網站資訊
- 網址：https://www.kindyinfo.com
- 名稱：幼園通 (KindyInfo)
- 描述：幼兒園地圖查詢系統，可根據類型、學生數、娃娃車、幼幼班、裁罰篩選

## 資料來源
- 資料來源：全國教保網
- 資料更新時間：2026/01/15（即時更新）

## 裁罰紀錄頁面
- URL：https://www.kindyinfo.com/blog/preschool-penalties
- 內容：包含最新幼兒園裁罰紀錄表格
- 欄位：日期、縣市、區域、幼兒園名稱、裁罰次數、裁罰內容

## 資料格式範例
從 HTML 解析出的資料格式：
- 日期：2025-02-21
- 縣市：新竹縣
- 區域：新豐鄉
- 幼兒園：新竹縣私立美德幼兒園
- 裁罰次數：2
- 裁罰內容：罰鍰：6,000元

## 整合建議
1. kindyinfo.com 的資料來源是「全國教保網」，與我們已經整合的 ap.ece.moe.edu.tw 是同一來源
2. 但 kindyinfo.com 提供了更友善的格式和歷史資料備份
3. 可以作為補充資料來源，特別是當全國教保網無法存取時

## 技術實作
- 網站使用 Nuxt.js (Vue) 框架
- 資料透過 Supabase 儲存
- 需要爬蟲解析 HTML 表格
